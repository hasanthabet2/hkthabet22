# Hasan Thabet - Professional Portfolio

A modern, dark-themed, and elegant front-end project showcasing a professional portfolio website.

## Features

- **Dark Theme**: Sleek black background with gold and neon blue accents.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Smooth Animations**: Fade-ins, slides, hover effects, and scroll-triggered animations.
- **Interactive Elements**: Hover effects on buttons, lists, and images.
- **Clean Code**: Well-organized HTML, CSS, and JavaScript with comments.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Transitions)
- JavaScript (ES6+)

## File Structure

- `index.html`: Home page with hero section, about, skills, tools, and vision.
- `contact.html`: Contact form page.
- `style.css`: Main stylesheet with dark theme and animations.
- `script.js`: JavaScript for interactivity (smooth scroll, scroll animations).
- `me.jpeg`: Profile image (ensure this file exists in the same directory).

## How to Run

1. Open `index.html` in a modern web browser.
2. Navigate using the fixed navigation bar.
3. Fill out the contact form on `contact.html`.

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Customization

- Change colors in `style.css` by modifying the CSS variables or color values.
- Add more sections by copying the `<section>` structure in `index.html`.
- Enhance animations in `script.js` using Intersection Observer.

## Performance Notes

- Uses modern CSS for smooth animations without heavy libraries.
- Optimized for fast loading with minimal assets.
- Responsive images and layouts for better performance on mobile.
